## Description
**Demonic Electron** application. This is **game-client**, which can find games for you , You can filter games by genre and platform.
## Tech Stack

 - **JavaScript**
 - **Electron**
 - **Electron Builder**
 - **NodeJS**
 
 - **ReactJS**
 - **Tailwind**
 - **SCSS**
 - **RawgAPI**
